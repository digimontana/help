Checking status of git
```
git status
```
Commit File
```
git commit -m "Max 50 characters"
```

Switch and create new branch, checking branch
```
git checkout -b new
git checkout
```
Merge branches
```
git merge master
```
Get changed files to local
```
git pull origin master
```
